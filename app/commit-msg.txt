fix: make EPUB drag-and-drop actually work on web

The previous DropZone relied on a .web.tsx platform file and passed
onDrop/onDragOver as JSX props on a <View>. react-native-web filters
unknown DOM handlers off View, so the drop events never reached the
underlying <div>, and the platform file wasn't picked up — so nothing
happened. Replaced with a single DropZone that gates on Platform.OS and
attaches the drag handlers via a ref + addEventListener on the real DOM
node. Also added a visible dashed drop area with a hint so it's
discoverable, with highlight on drag-over.
