feat: drag-and-drop EPUB on the import screen (web)

The import screen now accepts an EPUB dropped onto the file-picker area,
in addition to choosing one from a folder. Drag-and-drop is web-only
(DropZone.web.tsx; native uses a passthrough DropZone.tsx since OS file
drop isn't available there). A dropped File is a Blob, so it flows
through the same client-side parser as the picker — the parsing/branch
logic was extracted into a shared processBlob() called by both paths.
The publication file still never leaves the device.
